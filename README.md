# Code Metrics Analyzer

## Автор

**Галяветдинов Булат Фавиевич, группа М3113**

## Описание

Расширение для Visual Studio Code для анализа метрик кода. Плагин подсчитывает базовые характеристики файлов с исходным кодом.

## Возможности

### Анализ метрик файла

Команда: `Анализ метрик файла`

Анализирует открытый файл и показывает метрики:

- Общее количество строк
- Строки кода
- Комментарии
- Пустые строки
- Количество функций
- Количество классов
- Цикломатическая сложность

## Использование

### Способ 1: Контекстное меню

1. Откройте файл с кодом
2. Правый клик в редакторе
3. Выберите "Анализ метрик файла"

### Способ 2: Палитра команд

1. Нажмите `Cmd+Shift+P` или `Ctrl+Shift+P`
2. Введите "Анализ метрик"
3. Выберите команду

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Скомпилируйте: `npm run compile`
4. Запустите: нажмите F5 в VS Code
